# Employee-management-system
Employee Management System :
 Technologies Used
- Java
- JDBC
- MySQL

• Developed a console-based Employee Management System using Java and JDBC with MySQL database integration.
• Implemented CRUD operations such as Add, View, Update, Delete, and Search Employee functionalities.
• Applied OOP concepts, DAO architecture, exception handling, input validation, and PreparedStatement for secure database operations.
• Used MySQL Workbench for database management and GitHub for version control.
